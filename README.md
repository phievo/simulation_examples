# Example seeds

This repository contains the results of example simulations:
 - adaptation
 - adaptation_pruning
 - somite
 - somite_pruning
 - lacOperon
 - lacOperon_pruning
 - hox_pareto_light
 
 To download the result of a simulation on your computer, you can use phievo:
 
 ```python
import phievo
phievo.download_example_seed("adaptation")
 ```
 
 ## Hox pareto
